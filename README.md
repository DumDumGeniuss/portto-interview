# portto-interview

> For interview with Portto

## Setup for demo

```bash
# Node version v14.1.0
$ nvm install 14.1.0
$ nvm use 14.1.0

# Setp 1: install packages
$ yarn install

# Step 2: Build
$ yarn build

# Step 3: Start running it
$ yarn start
```
